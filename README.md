## Anaconda
### anaconda install
Anaconda website: https://www.anaconda.com/download/<br>
Download Python 3.6 linux version, and run command.
```
sh ANACONDA_FILE
```
Using command ```which python``` or ```which python3``` to check using Anaconda python or not.
